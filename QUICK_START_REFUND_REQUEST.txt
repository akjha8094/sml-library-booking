🚀 HOW TO TEST (3 SIMPLE STEPS):

┌─────────────────────────────────────────────────────────────────┐
│  STEP 1: Test as User                                           │
└─────────────────────────────────────────────────────────────────┘

  1. Open browser: http://localhost:3000
  2. Login as user
  3. Go to "My Bookings"
  4. Click "Request Refund" button on any active booking
  5. Fill the form and submit
  6. Go to "My Refund Requests" to see your request

┌─────────────────────────────────────────────────────────────────┐
│  STEP 2: Test as Admin                                          │
└─────────────────────────────────────────────────────────────────┘

  1. Open browser: http://localhost:3000/admin
  2. Login as admin
  3. Click "Refund Requests" in sidebar
  4. See the user's request in the table
  5. Click "Review" button
  6. Approve or Reject with notes
  7. Submit decision

┌─────────────────────────────────────────────────────────────────┐
│  STEP 3: Verify Auto-Processing                                 │
└─────────────────────────────────────────────────────────────────┘

  1. After admin approves, switch back to user
  2. Check notifications (should show approval)
  3. Go to "Wallet" page
  4. See refund amount credited
  5. Go to "My Refund Requests"
  6. Status should show "Completed"
